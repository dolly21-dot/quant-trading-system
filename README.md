## 🚀 快速开始

### 1. 安装依赖
```bash
pip install -r requirements.txt
```

### 2. 配置
```bash
cp .env.example .env
# 编辑 .env 填入:
# - T212_API_KEY: Trading 212 API密钥
# - 其他数据源key (可选)
```

### 3. 回测验证
```bash
python main.py --action backtest --symbol AAPL --strategy momentum_trend
```

### 4. 启动系统
```bash
# 模拟环境 (强烈建议先在demo环境验证)
python main.py --mode demo --action start

# 实盘环境 (确认策略有效后)
python main.py --mode live --action start
```

## ⚠️ 风控硬约束

| 约束 | 值 | 说明 |
|------|-----|------|
| **不做杠杆** | 🚫 | 系统硬编码，禁止CFD/保证金交易 |
| 单股最大仓位 | 10% | 单只股票不超过组合10% |
| 单笔最大风险 | 2% | 单笔交易最大亏损不超过账户2% |
| 默认止损 | 5% | 未设止损自动添加5%止损 |
| 默认止盈 | 10% | 未设止盈自动添加10%止盈 |
| 日最大回撤 | 3% | 超限触发交易暂停（熔断） |
| 单行业最大 | 30% | 同行业敞口不超过30% |

## 📊 策略说明

### 动量趋势策略 (momentum_trend)
- **入场**: EMA金叉 + ADX趋势确认 + RSI非超买
- **出场**: EMA死叉 / RSI极端值 / ATR止损
- **仓位**: ATR波动率调仓
- **适用**: 趋势明确的市场

### 均值回归策略 (mean_reversion)
- **入场**: 布林带触轨 + RSI超买超卖 + Z-Score极端
- **出场**: 价格回归中轨 / 最大持仓时间
- **仓位**: 保守仓位（8%上限）
- **适用**: 震荡盘整市场

### 事件驱动策略 (event_driven)
- **入场**: 重大新闻 + 情绪极端 + 成交量异动
- **出场**: 事件消化 / 止损
- **仓位**: 最小仓位（6%上限）
- **适用**: 财报季、重大政策发布

## 🧠 自反思机制

系统每日收盘后自动执行复盘：

1. **绩效评估** → 计算Sharpe、胜率、盈利因子、最大回撤
2. **策略评级** → A/B/C/D/F 五级评定
3. **失败诊断** → 分析亏损交易原因
4. **市场环境** → 判断牛市/熊市/震荡，匹配策略
5. **自动调参** → C级策略谨慎微调（幅度≤20%）
6. **策略禁用** → D/F级策略自动暂停

## 📅 调度任务

|------|------|-----------|

| 持仓监控 | 盘中每10分钟 | 14:30-21:00 |
| 盘后同步 | 每日 | 21:30 |

| 周度复盘 | 每周日 | 22:00 |

## 📌 Trading 212 API 适配

- **认证**: API Key (Header: Authorization)
- **环境**: Demo(模拟) / Live(实盘) 独立隔离
- **订单类型**: 市价单、限价单、止损单、止损限价单
- **数据**: 持仓、挂单、历史交易、分红、CSV报告
- **Python SDK**: `t212-api` (推荐) 或 `httpx` 直接调用
